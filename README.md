# Weathendar
**A productivity tool built with modern technologies**\
Weathendar is a productivity tool that bundles crucial day to day information into one place. It has a calendar, weather display, and motivation display that features quotes and my dog Oach.

## To run or contribute 
`npm run dev` for running dev server
`npm run build` for creating a production build
`npm run start` for testing production build
